🚀 3步快速使用
 
1. 下载工具：点击Releases页面，下载 KiraraProxy-Setup-v1.0.0.exe ，双击安装；
2. 添加软件：打开工具→进入“快递单配置”→点击“添加软件”，选择要代理的.exe文件+切换模式；
3. 启动代理：点击“绮良良出发”按钮（或按Ctrl+K），软件自动运行，卡死会自动重启。
